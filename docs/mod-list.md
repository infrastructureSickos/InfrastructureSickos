# InfrastructureSickos Mod List

All 3rd-party mods must be available on Modrinth for 1.20.1 Forge.

## Core Tech Mods (Progression Order)

| Mod | Role | Modrinth |
|-----|------|----------|
| Create | Tier 1 — Rotational power, mechanical crafting, trains | Yes |
| Immersive Engineering | Tier 3 — Industrial machines, multiblocks, power | Yes |
| Botania | Tier 4 — Mana-based nature magic, endgame | Yes |
