# infrastructureSickos (GitHub Organization)

This directory holds local clones of repos from the `infrastructureSickos` GitHub organization.

- `InfrastructureSickos/` — Custom Forge mod source code (mirrors github.com/infrastructureSickos/InfrastructureSickos)
- `server/` — Minecraft server config and KubeJS scripts (mirrors github.com/infrastructureSickos/server)
