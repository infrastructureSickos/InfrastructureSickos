# Create — Tweaks

## Recipe Modifications (via KubeJS)

### Treated Wood Requirement
- **All Create recipes that use wood planks/logs** are replaced to use **IE Treated Wood** instead
- This includes: shafts, cogwheels, mechanical bearings, windmill bearings, water wheels, casings, etc.
- Players must build an IE Coke Oven and produce creosote oil before starting Create
- Creates early-game interplay between Create and IE rather than strict sequential progression

### Water Wheel & Windmill Bearing Improvements
- **Water Wheel** recipe modified to use **treated wood** + **treated wood rods** (borrowing IE water wheel materials)
- **Windmill Bearing** sails modified to use **wool** or **hemp fabric** (IE's windmill sail material)
- These are the player's first power source — treated wood makes them feel sturdy and industrial

### Steel Requirements
- **Crushing Wheels** — recipe modified to require **IE Steel**
- **Train components** (tracks, stations, controls, casings) — recipes modified to require **IE Steel**
- **Steam Engine** — recipe modified to require **IE Steel**
- This gates Create's late game behind IE steel production, reinforcing the Create ↔ IE interplay
