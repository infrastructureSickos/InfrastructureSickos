# CC: Tweaked — Tweaks

## Recipe Replacements (via KubeJS)

All CC: Tweaked crafting recipes replaced with expensive Create Mechanical Crafting recipes using IE-tier resources.

### Materials Used
- **Aluminum** (Immersive Engineering)
- **Steel** (Immersive Engineering)
- **HV Wire/Components** (Immersive Engineering)
- **Treated Wood** (Immersive Engineering)
- **Brass** (Create)
- **Precision Mechanisms** (Create)

### Specific Recipes TBD
- Computer → Mechanical Crafting recipe using aluminum casing, brass, precision mechanisms
- Turtle → Mechanical Crafting recipe using computer + additional IE/Create components
- Monitor → Mechanical Crafting recipe using aluminum, glass, redstone
- Disk Drive, Printer, Speaker, etc. — all similarly gated

### Intent
ComputerCraft is powerful automation. Gating it behind expensive Create + IE recipes keeps it as a mid-to-late game reward. Aluminum specifically ties it into the IE progression tier.
