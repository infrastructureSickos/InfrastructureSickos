# Ultimate Plane Mod — Tweaks

## Recipe Modifications (via KubeJS)

### Plane Body
- Replace default body materials with **IE colored sheetmetal**
- Plane color matches sheetmetal color

### Fuel
- Uses **UC Gas Station** (shared with cars), which accepts **IP diesel**
- Remove standalone furnace fuel support — planes need diesel, not coal

### Frames
- Plane frames use **IE steel**

### Assembly
- **Remove crafting table recipes** for planes
- Plane components crafted via **Create Sequenced Assembly** and **Mechanical Crafting**
- Final plane assembly via **Mechanical Crafting**
- Use **Create stash** instead of vanilla chest in any storage-related recipes

### Wheels (Sequenced Assembly)
- Same wheel recipe as Ultimate Car Mod (iron sheet + bitumen + press)

### Engines (Sequenced Assembly)
- Replace Diamond Reinforced Iron with **steel** components
- Plane engine via Sequenced Assembly line

### Tanks
- Replace mod's fuel tanks with **Create Fluid Tanks**

### Other Recipe Changes
- Wood plank bodies replaced with **IE treated wood**
- Fuel uses IP diesel/kerosene
