# Aleki's Nifty Ships — Tweaks

## Recipe Modifications (via KubeJS)

### Materials
- Hull planks → **IE Treated Wood** (creosote-soaked = waterproofed)
- Sails → **Wool** (plain, no processing needed)
- Mast → **Treated Wood** (tall, sturdy)
- Any iron fittings → **Iron** (vanilla, no brass)
- Rudder → Treated Wood + Iron
- Any storage → vanilla **Chest**

### Intent
Ships are the earliest transport option. The only gate is the Coke Oven (for treated wood). No Create machines, no brass, no fancy processing. Build a coke oven, treat some wood, and set sail.
